How it works

Three steps. Then you forget it's there.

  1. 1

    Connect Jira and/or Microsoft 365

    Sign in with an Atlassian API token, Microsoft 365 (OAuth), or both. Credentials stay in your macOS Keychain.

  2. 2

    Tell Nowtify what's urgent to you

    Per-trigger watchlists. Pick which users' tickets count for SLA, which colleagues are Teams VIPs, which senders' emails matter.

  3. 3

    A pulse at the edge of your screen when it fires

    No popups. No sounds. No dock badge. Just a thin colored glow you can look at when you want to.

What it watches

Six things Nowtify can watch.

Turn on what's relevant. Each trigger has its own color, its own watchlist, and its own threshold.

  • Major Incident

    Any open ticket flagged as a Major Incident across your Jira instance. Fires instantly, clears when the flag drops or you dismiss it.

  • SLA breach imminent

    Watched users' tickets whose SLA remaining time drops under a threshold you set (e.g. 30 minutes). Per-condition color.

  • SLA breached

    Watched users' tickets whose SLA cycle has already expired. Stays lit until you dismiss.

  • My pending approvals

    JSM service-desk requests waiting on your approval. Counts only what's assigned to you, not the whole queue.

  • Teams VIP messages

    Unread Teams chat messages from people you've marked as VIPs. Threshold configurable. Clears on read.

  • Outlook email from watched senders

    Unread Outlook emails from specific people. Same VIP pattern as Teams. Clears on read.

FAQ

Install, security, and everything else.

How do I install it?

Nowtify is not Apple-code-signed (we're not paying $99/yr for a free tool yet), so macOS Gatekeeper will block the first launch. Here is the one-time bypass:

  1. Download the .dmg and double-click to mount it.
  2. Drag Nowtify into Applications.
  3. Right-click Nowtify.app in Applications, choose Open, then click Open in the warning dialog.
  4. If that dialog does not give you an Open button, instead open System Settings → Privacy & Security, scroll down, and click Open Anyway next to the Nowtify message.
!

"Nowtify" cannot be opened because Apple cannot check it for malicious software.

This software needs to be updated. Contact the developer for more information.

↑ The dialog in System Settings → Privacy & Security. Click "Open Anyway."

After the first launch, auto-update handles every future version. You only do this once.

What do I connect it to?

Atlassian (Jira or Jira Service Management) via an API token, and/or Microsoft 365 via OAuth (Teams chat + Outlook mail). You can use just one or both.

Do I need admin permissions?

Atlassian: just your own user and an API token (create one at id.atlassian.com → Security → API tokens).

Microsoft 365: the first time you sign in, an admin in your tenant has to consent to the Nowtify Entra app once. This has already been done for the internal employee audience this site targets. External users would need their own admin's consent (the current Entra app is single-tenant; multi-tenant is on the roadmap).

Is this safe to install?

Yes. Unsigned does not mean malicious - it just means we haven't paid Apple's Developer Program fee. Every line of source is public on GitHub. The app makes outbound HTTPS calls only to *.atlassian.net and graph.microsoft.com (plus GitHub for update checks). No remote command-and-control, no telemetry, no shared inbox.

What does it do with my data?

Everything stays local. Atlassian API tokens and Microsoft refresh tokens are encrypted in your macOS Keychain via Electron's safeStorage. Polling happens from your laptop directly to Atlassian and Microsoft. There is no Nowtify backend, no telemetry, no analytics.

Why isn't it code-signed?

Apple's Developer ID is $99/year. For a free internal tool, that hasn't been justified yet. If usage grows or the friction of the Gatekeeper bypass becomes a real blocker, we'll revisit.

Windows or Linux?

macOS only. The peripheral-pulse UX is built on macOS-specific window-level APIs (transparent, click-through, always-on-top windows per display). No roadmap commitment for other platforms.

How do I uninstall?

Drag Nowtify.app from /Applications to the Trash, then optionally delete ~/Library/Application Support/Nowtify to remove your config.
